# Assets Folder

Place your app assets in this folder:

## Required Images

- **icon.png** - App icon (1024x1024 px)
- **splash.png** - Splash screen (1284x2778 px)
- **adaptive-icon.png** - Android adaptive icon (1024x1024 px)
- **favicon.png** - Web favicon (48x48 px)

## Temporary Usage

For development, you can use placeholder images or the app will use Expo's default assets.

To generate proper assets, you can:
1. Create your own images with the sizes above
2. Use online tools like https://www.appicon.co/
3. Use design tools like Figma, Sketch, or Adobe XD
